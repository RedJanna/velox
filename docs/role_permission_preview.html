Kullanım notu

Bu ekran, rol tanımı yaparken yöneticinin neyi açıp kapattığını açıkça görmesi için tasarlanmıştır.

  • Sayfa erişimi kapalıysa ilgili pencere menüde görünmez.
  • İşlem izni kapalıysa kullanıcı butonu görse bile API işlemi reddetmelidir.
  • Kritik izinler ayrıca yeniden doğrulama gerektirebilir.

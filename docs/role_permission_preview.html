Kullanım notu

Bu ekran, rol tanımı yaparken yöneticinin neyi açıp kapattığını açıkça görmesi için tasarlanmıştır.

  • Sayfa erişimi kapalıysa ilgili pencere menüde görünmez.
  • İşlem izni kapalıysa kullanıcı butonu görse bile API işlemi reddetmelidir.
  • Kritik izinler ayrıca yeniden doğrulama gerektirebilir.

Güvenlik sınırı

Bu önizleme yalnızca arayüz davranışını gösterir. Kalıcı yetkilendirme kararı backend API, otel kapsamı ve kayıt düzeyi kontrollerle verilmelidir.

Yeni kullanıcı oluştur

Kullanıcı adı, geçici parola, rol (yetki seviyesi), departman (organizasyon birimi) ve zorunlu 2FA tercihini tek adımda tanımlayın.

Önizleme modunda parola saklanmaz. Backend entegrasyonunda parola hashlenmeli, 2FA sırrı ayrı güvenli alanda oluşturulmalıdır.
